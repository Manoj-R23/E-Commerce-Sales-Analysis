# E-Commerce Sales Analysis
## Pluto Academy Data Analytics Internship — Project 1

---

## 📌 Project Overview
Analysed 12 months of Brazilian e-commerce sales data 
from Olist dataset to identify trends, compare categories 
and regions, and provide actionable business recommendations.

---

## 🛠️ Tools Used
- **Python** (Pandas, Matplotlib, Seaborn)
- **Power BI** (Interactive Dashboard)

---

## 📁 Dataset
- Source: Olist Brazilian E-Commerce Dataset (Kaggle)
- Size: 99,441 orders across 7 tables

---

## 📊 Analysis Questions Answered
1. Which product category has highest revenue?
2. Which month had peak sales?
3. Which region performs best?
4. What is the average order value trend?
5. What is the customer review score distribution?

---

## 🔍 Key Findings
1. **Health & Beauty** is the top revenue category — R$ 1,237,439
2. **November 2017** had peak sales — 7,288 orders (Black Friday)
3. **São Paulo (SP)** dominates regional sales — R$ 5,091,551
4. Average order value is stable between **R$ 108 — R$ 132**
5. **76.81%** customers gave 4 or 5 star ratings

---

## 📈 Dashboard
Built an interactive Power BI dashboard showing:
- Total Revenue: R$ 13.28M
- Total Orders: 96,455
- Top Category: Health & Beauty
- Best Month: November 2017
